🚀 Deploy to GitHub Pages

Your app gets a permanent public URL — e.g. https://yourname.github.io/logistics-ke/ — that anyone can open on any phone. No hosting cost.
1

Create a GitHub account

Go to github.com → Sign up. Any email works. Verify your email when the confirmation arrives.

2

Create a repository

Click + → New repository. Name it logistics-ke. Set visibility to Public. Tick "Add a README file". Click Create repository.

3

Upload and rename the file

Click Add file → Upload files. Drop in logistics-ke.html. After upload, open it in the repo, click ✏️ Edit, and rename it to index.html. Commit.

4

Turn on GitHub Pages

Settings → Pages → Source → Deploy from a branch. Branch: main, folder: / (root). Click Save. Your URL appears in ~60 seconds.

5

Share your link

Share https://YOUR-USERNAME.github.io/logistics-ke/ via WhatsApp or SMS. Everyone who opens it is on the same live database.

Your admin passphrase is in the file. Don't share it. Change it in the file before uploading if others may see the source code.
6

Updating the app

Download the latest file. Open index.html in your GitHub repo → ✏️ Edit → Select all → Paste new content → Commit. Live in ~30 seconds. Your data in the Sheet is untouched.
